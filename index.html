Primeiro Contato

Apesar de não estar diretamente ligado à área de programação, sempre busquei soluções tecnológicas para resolver problemas no meu papel como líder de produção.
Meu primeiro contato com uma linguagem de programação foi através do VBA no Excel. Na ocasião, nossa equipe precisava documentar as alterações nos processos produtivos, visando consultas futuras, correções e sistematização. Desenvolvi um projeto em VBA com uma interface intuitiva e tarefas automatizadas.

Fora da Caixa

Após essa experiência, decidi me aprofundar na área de desenvolvimento e programação, o que me levou a descobrir o desenvolvimento web. Continuo dedicando-me a conhecer novas técnicas e linguagens e a cada estudo, fico mais impressionado com as possibilidades que essa área pode oferecer.

CSS

GIT

Desde o início da minha jornada como desenvolvedor, tenho usado o Git para versionar meus projetos. Optei por migrar para o Linux para me familiarizar com o terminal, o que tem contribuído significativamente facilitando minhas tarefas de desenvolvimento.

SASS CSS

Com um sólido conhecimento em Sass, incluindo técnicas avançadas como Nesting, Partials, Variáveis, Ampersand, Mixins e Extend, considero-me capacitado para criar e estruturar folhas de estilo CSS de forma eficaz e organizada.

TAILWIND CSS

Com sólido conhecimento em Tailwind CSS, considero-me capacitado a criar designs responsivos de forma prática e eficiente. Domino a abordagem mobile-first nativa do Tailwind, a aplicação de seus componentes pré-definidos e a capacidade de estender e personalizar os estilos conforme as necessidades do projeto, resultando em uma experiência de desenvolvimento ágil e um código CSS modular e legível.

REACT

Possuo sólido conhecimento em React, dominando o uso de Hooks, implementação de gerenciamento de estado global com Context API, aplicação do React Router para roteamento dinâmico e técnicas avançadas de performance, garantindo aplicações rápidas e escaláveis.

FIGMA

À medida que avanço nos estudos de programação, aprimorei minhas habilidades no uso da ferramenta Figma. Agora, sinto-me confortável em criar layouts consistentes e adaptáveis, fazendo uso de componentes, auto layout e constraints para otimizar o desenvolvimento web.
